# Minas Silk Collection Website

A luxury ultra-premium bridal fashion website for **Minas Silk Collection**, Tirur, Kerala.

This project is designed with a cinematic and editorial-style UI focused on luxury silk collections, bridal experiences, Kerala traditions, and modern fashion storytelling.

---

## ✨ Features

- Ultra luxury UI/UX design
- Fully responsive layout
- Smooth scroll animations
- Interactive custom cursor
- Bridal collections showcase
- Occasion-based fashion sections
- Kerala heritage-inspired aesthetics
- Elegant typography using Google Fonts
- Horizontal bridal story slider
- Animated hover effects
- Scroll reveal animations
- Modern premium dark theme

---

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/riyaspullurofficial/MinasSilkColectionWebsiteHTML.git
```

2. Open project folder

```bash
cd MinasSilkColectionWebsiteHTML
```

3. Run the project

Simply open:

```bash
index.html
```

in your browser.
